Installation

For Linux/[macOS](https://apple.com/macos) run the following command. ```bash command here ``` For [NT](https://en.wikipedia.org/wiki/Windows_NT) head [here](https://github.com/HttpAnimation/StratumOS/tree/stable/Builds) to download.


/host

The /host command allowing you to run a command on the host system. ```bash >>/host echo "Hello world" Hello world ```


/clear

The /clear command clears the display. ```bash >>/clear ```


/say

The /say command will repeat text ```bash >>/say I like boys :3 I like boys :3 ```
